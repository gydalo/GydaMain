HotView Labs

HotView Labs is a simulated blog page for a tech firm, focusing on the latest tech gear. It includes login and registration features, so logged-in users can edit, delete, and post new content, while non-logged-in users can read the posts. This project was quite challenging, but I enjoyed it a lot. It made me learn more about using JavaScript and how to combine JavaScript, HTML, and CSS.
